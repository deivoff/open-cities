# Open Cities Project

## Docker

Для разработки необходимо развернуть контейнер с базой данных и применить миграции. Для этого выполните следующие команды:

1. Перейти в папку **docker**:

        cd docker

2. Запустить **docker-compose**:

        docker-compose up -d
